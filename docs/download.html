Prerequisite

TNSG uses many features of the Fortran 2003 standard, and can be successfully compiled by GNU fortran compiler later than 5.4.0 version. In Debian/Unbuntu, it can be installed by

apt-get install gfortran

TNSG uses MPI for parallel programming. A popular choice is Openmpi. In Debian/Ubuntu, it can be installed by

apt-get install libopenmpi-dev

TNSG uses Lapack and Blas as math library. In Debian/Unbuntu, they can be installed by

apt-get install liblapack-dev

Or you may use Intel Math Kernel Library which integrets Lapack and Blas.

Installation Guide

TNSP can be installed with make. Configuration have to be set in the make.inc file. A make.inc for a Linux or Mac OS running GNU compilers is given in the main directory. Then the user can do "make" in the current directory to build the standard library "output/TNSG_x"(x for version), and do "make test" to run the test PEPS program.

Using TNSG in Your Program

Before using TNSG, the module should be used

use mod_name

To use the TNSP in your code, you should use the following option when compiling your code

-I$(TNSG)/TNSG_x/include

and the following option to link your code with the TNSP

-L$(TNSG)/TNSG_x/lib -lib_name

, where $(TNSG) is the TNSG directory and x is the version. And make sure you have linked TNSG with the math library.

For avaliable mod_name and lib_name, please visit our document page.
